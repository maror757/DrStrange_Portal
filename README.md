# DrStrange Portal
Python script for creating the Doctor Strange portal effect in Maya
### Arnold Renderer
Make these changes in Maya
* Render Settings >> Arnold Renderer >> Motion Blur >> Enable
* Arnold >> Lights >> Skydome Lights
